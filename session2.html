The Seven Wastes of Software Development

  2. Extra Features
  3. Re-learning

  6. Delays

Steps to Remove Waste

  1. Define Problem/Gather data
  2. Analyse the Situation
  3. Create a hypothesis
  4. Perform Experiments
  5. Verify Results
  6. Share/Standardise/Create Policies
